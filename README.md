## Stacks utilizadas

**Front-end:** React, ContextAPI, MUI

**Back-end:** Node, Mongoose


## Funcionalidades

- Cadastro e login de usuários com autenticação
- Cadastro e edição das vacinas
- Pdf das vacinas cadastradas
- Filtro das vacinas
